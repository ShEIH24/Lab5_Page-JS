# Page-JS - Структура проекта

## Описание
Проект содержит 7 интерактивных JavaScript компонентов, реализованных на чистом JavaScript без использования сторонних библиотек и фреймворков.

## Структура файлов

### Стили
- **`style.css`** - единый файл стилей для всех компонентов, включает общие стили и специфические стили для каждого компонента

### Отдельные HTML файлы компонентов

#### `accordeon.html`
отвечает за компонент аккордеона с раскрывающимися секциями

#### `animation.html`
отвечает за демонстрацию различных анимаций при наведении курсора

#### `form_validation.html`
отвечает за форму с валидацией полей ввода в реальном времени

#### `progress_bar.html`
отвечает за прогресс-бар с возможностью увеличения и уменьшения значения

#### `switch_text.html`
отвечает за переключатель размера шрифта текста

#### `timer.html`
отвечает за таймер обратного отсчёта с функциями старт, пауза и сброс

#### `to-do_list.html`
отвечает за список задач с возможностью добавления, отметки выполнения и удаления


## Принцип организации
Все компоненты используют общий файл `style.css`, что обеспечивает единообразие стилей.

## Технологии
- HTML5
- CSS3
- JavaScript
